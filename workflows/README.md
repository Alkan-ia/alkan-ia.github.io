# Workflows Documentation

This directory contains documentation for all n8n workflows created for the AI & Automation Agency.

## Structure

- **Root (`/workflows`)**: Individual workflow documentation files
- **`/diagrams`**: Mermaid diagrams and visual representations of workflows

## Naming Convention

Each workflow documentation file should follow this format:

```
[category]-[action]-[target].md
```

**Examples:**

- `lead-gen-capture-form-to-crm.md`
- `marketing-schedule-social-posts.md`
- `ai-generate-content-gpt.md`

## Documentation Template

When creating a new workflow, include:

1. **Overview** - Purpose and use case
2. **Mermaid Diagram** - Visual flow representation
3. **Configuration** - Required credentials and settings
4. **Inputs/Outputs** - Expected data format
5. **Testing Checklist** - Validation steps
6. **Maintenance Notes** - Known issues and updates

---

*Workflows will be documented here as they are created.*
